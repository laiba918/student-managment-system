# Student Management System

A Python-based Student Management System developed to perform basic CRUD operations and manage student records efficiently.

## Features
- Add student records
- View all students
- Edit student details
- Delete student records

## Technologies Used
- Python
- Lists / Dictionaries
- Functions
- File Handling (optional future upgrade)
